# Mud Pit Canada Route Notes

This app uses a distinct industrial `wmc-*` visual system and spreadsheet-driven pSEO pages.

- Landing page source data lives in `lib/watermain-data.ts`.
- Dynamic service and city pages are served from `app/[...slug]/page.tsx`.
- Use one primary keyword target per landing page.
- Keep copy visitor-facing: describe site conditions, access, equipment, disposal, safety, and next steps.
- Preserve JSON-LD for Organization or LocalBusiness, Service, FAQPage, and BreadcrumbList.
