# Mud Pit Canada

Programmatic SEO Next.js site generated from `pseo_mud_pit_cleaning_canada.csv`.

## Pages

- Home: `app/page.tsx`
- Service index: `app/services/page.tsx`
- Generated service and city pages: `app/[...slug]/page.tsx`
- Spreadsheet-derived data: `lib/watermain-data.ts`

The CSV currently produces 313 landing pages: service hubs plus nested location pages such as `/mud-pit-cleaning/toronto-on`.
